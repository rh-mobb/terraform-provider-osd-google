# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 0.x    | :white_check_mark: |

We provide security updates for the latest minor version. Upgrade to the latest release to receive fixes.

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please report it responsibly:

1. **Do not** open a public GitHub issue for security vulnerabilities.
2. Report via one of these channels:
   - **GitHub Security Advisories**: Open a [private security advisory](https://github.com/rh-mobb/terraform-provider-osd-google/security/advisories/new) on the repository.
   - **Red Hat**: If you have a Red Hat support relationship, use your usual channel for security issues.
3. Include a clear description of the vulnerability, steps to reproduce, and potential impact.
4. We aim to respond within 5 business days and will keep you informed of our progress.
5. We ask for reasonable time to address the issue before public disclosure.

Thank you for helping keep this project and its users safe.
